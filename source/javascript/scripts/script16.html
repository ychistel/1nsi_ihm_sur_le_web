Interagir avec le document en JAVASCRIPT

Le script de cette page doit modifier les règles CSS de certains éléments HTML selon les événements de la souris et du clavier.

Quelques événements reconnus de la souris :

  • click : on clique sur l'élément ciblé;
  • mousedown : appui sur la souris, clic non relaché ;
  • mouseup : relachement de la souris après un clic;
  • mouseover : on survole l'élément ciblé;
  • mouseout : on ne survole plus l'élément ciblé.

Quelques événements reconnus du clavier :

  • keydown : un appui sans relacher d'une touche du clavier;
  • keyup : relachement de la touche du clavier;
  • keypress : frapper sur une touche du clavier

Votre travail:

  1. Vous devez éditer ce fichier script16.html et compléter le script JAVASCRIPT pour :
  2. Encadrer le titre s'il est cliqué
  3. Afficher un fond bleu si la première liste est survolée.
  4. Afficher un fond vert sur la seconde liste s'il est survolé et revenir au fond blanc s'il n'est pas survolé.
  5. Afficher un fond rouge si du texte est saisi dans la zone texte.
