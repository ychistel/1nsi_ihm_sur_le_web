Modifier des règles CSS avec JAVASCRIPT

Le script de cette page doit modifier les règles CSS de certains éléments HTML. Il faut au préalable avoir une connaissance de la structure HTML du document en repérant les id, class et sélecteurs utilisés.

Lorsque le sélecteur est atteint, on enchaine avec la méthode style en lui attribuant les nouvelles valeurs CSS:

Par exemple:
  • Changer la couleur : document.getElementById('id').style.color = "couleur";
  • Changer l'alignement : document.getElementById('id').style.textAlign = "alignement";
  • Ajouter une bordure : document.getElementById('id').style.border = "valeurs";
Remarque:

Les propriétés CSS sont parfois écrites avec un trait d'union. En javascript, les propriétés ne sont pas écrites avec un trait d'union. Celui-ci est remplacé par la majuscule du mot situé après le tiret. Par exemple, la propriété CSS padding-left sera écrite paddingLeft en JAVASCRIPT.

Votre travail:

  1. Vous devez éditer ce fichier script14.html et compléter le script JAVASCRIPT.
  2. Centrer le titre en rouge;
  3. Le paragraphe d'introduction sera mis en couleur bleue.
  4. Le second paragraphe sera sur un fond gris clair.
  5. Les puces de la liste d'exemples seront remplacées par des carrés.
  6. Le contenu de la remarque sera en italique.
  7. Le titre "Votre travail" sera encadré.
  8. La liste des tâches à réaliser aura un espacement de lettres égal à 5px (difficile).

Si votre script est correct, en actualisant cette page, vous devez voir les modifications du document.
