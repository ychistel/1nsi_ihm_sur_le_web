Les formulaires avec JAVASCRIPT

Le script de cette page doit afficher un message de bienvenue avec le nom et le prénom du lecteur qui auront été saisis dans un formulaire.

Votre travail:

  1. Vous devez éditer ce fichier script18.html et compléter le code HTML pour avoir un formulaire contenant :
    • Le prénom du lecteur dans un élément input
    • Le nom du lecteur dans un élément input
    • Un bouton pour soumettre le formulaire.
  2. Vous devez compléter le script JAVASCRIPT pour récupérer les valeurs et les afficher dans un message de bienvenue.

Si votre script est correct, en actualisant cette page, vous devez voir votre message de bienvenue écrit ci-dessous.
