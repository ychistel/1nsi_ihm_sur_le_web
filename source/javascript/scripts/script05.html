Les structures conditionnelles en JAVASCRIPT

Une structure conditionnelle en JAVASCRIPT proche de celle utilisée en langage PYTHON est de la forme:
if (condition) { instructions; } else { instructions; } avec un else facultatif.

Le script de cette page soumet une question au lecteur. Un test est effectué pour afficher une réponse différente selon les cas.

<script type="text/javascript">
let nombre = prompt('Donner un nombre compris entre 1 et 10 ?');
if (nombre > 1 && nombre < 10) {
    console.log("Vous avez saisi "+nombre+". Vous avez respecté la consigne !");
}
else {
    console.log("Vous avez mal lu la consigne ou vous ignorez ce qu'est un nombre compris entre 1 et 10 !");
}
</script>

Commentaires:

  • Une fenêtre de saisie permet de donner le nombre avec la fonction prompt.
  • Le nombre est enregistré dans une variable nombre avec la commande let.
  • La condition est double avec un ET symbolisé par && ce qui signifie que l'instruction est réalisée si le nombre est à la fois plus grand que 1 et plus petit que 10.
  • Le else donne une alternative si la condition n'est pas vérifiée.
