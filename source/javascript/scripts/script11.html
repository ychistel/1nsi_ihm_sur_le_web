Modifier le contenu d'un document HTML avec JAVASCRIPT

Les programmes réalisés jusqu'à présent sont peu utiles pour le document web qui les appelle. Le véritable intérêt de JAVASCRIPT est de pouvoir interagir avec le document WEB pour le modifier.

On peut modifier directement le document WEB au moment de son chargement.
Pour modifier un document WEB, il faut :

  1. Créer une fonction qui réalisera la tâche demandée
  2. Identifier les éléments HTML qui seront modifiés par la fonction
  3. Appeler la fonction dans un script situé en bas de page

Le script de cette page utilise une fonction pour calculer le carré d'un nombre saisi par le lecteur de la page. Maintenant, 3 parties du document WEB sont utilisés !

Le carré du nombre 1 est égal au nombre 1.

Le script JAVASCRIPT dans le <head> du document
<script type="text/javascript">
function carré(){
    let nombre = prompt("Saisir le nombre dont vous voulez calculer le carré:");
    let monCarré = nombre * nombre ;
    document.getElementById('nombre').innerHTML = nombre;
    document.getElementById('carré').innerHTML = monCarré;
}
</script>
Le code HTML dans le <body> du document qui recevra les valeurs à afficher
</body>
...
<p>Le carré du nombre <span id="nombre">1</span> est égal au nombre <span id="carré">1</span>.</p>
...
</body>
Le script JAVASCRIPT à la fin du document WEB pour lancer l'exécution du script appelé.
</body>
...
<script> carré(); </script>
</body>

Commentaires:

  • On crée une fonction carré() sans paramètre qui sera appelée à la fin du chargement de la page web.
  • On utilise encore une fenêtre de saisie avec la fonction prompt.
  • La valeur est enregistrée dans une variable nombre avec la commande let.
  • On utilise la fonction getElementById('id') pour repérer la partie du document web à modifier.
  • On enchaine avec la fonction innerHTML pour modifier la valeur à afficher.
  • Le document WEB est chargé, la fonction carrée est appelée et les deux valeurs sont modifiées aussitôt !
