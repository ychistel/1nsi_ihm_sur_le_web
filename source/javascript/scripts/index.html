Langage JAVASCRIPT

JAVASCRIPT est un langage de programmation orienté objet. Cela signifie que les fonctions s'appliquent sur des objets identifiés. Les fonctions s'appellent des méthodes.

  • La fenêtre du navigateur est un objet JAVASCRIPT nommé window qui possède ses fonctions et variables.
  • Le document web est un objet document qui possède aussi ses fonctions et ses variables.

Cela a pour conséquence de faire précéder chaque méthode par le nom de l'objet séparés par un point.
L'introduction d'un script JAVASCRIPT, se fait en général dans l'entête du document HTML entre les balises <head> et </head>.
Le script est écrit entre les balises <script> et </script>.

Vous allez parcourir un ensemble de pages HTML pour découvrir la programmation en JAVASCRIPT. Les fichiers se nomment scriptXX où XX est un nombre entier compris entre 1 et 18 inclus. L'accès se fait par le lien situé en bas du document.
Ces fichiers se trouvent dans le dossier script. Certains d'entre eux seront à compléter avec un éditeur de texte comme notepad++ ou le bloc note. Les scripts que vous allez écrire restent simples et ont pour objectif la découverte de JAVASCRIPT.

Bon courage à vous.
