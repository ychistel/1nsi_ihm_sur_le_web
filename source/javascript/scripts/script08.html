La boucle for en JAVASCRIPT

Le script de cette page doit demander un nombre entier positif et afficher dans la console la somme de tous les nombres entiers positifs qui lui sont strictement inférieurs.

Votre travail:

  1. Vous devez éditer ce fichier script08.html et compléter le script JAVASCRIPT.
  2. Le nombre sera à saisir par le lecteur.
  3. Une variable somme sera initialisée à 0.
  4. Une boucle réalisera l'addition de la somme et du compteur de boucle.
  5. La somme sera affichée dans la console.

Si votre script est correct, en actualisant cette page, vous devez voir votre valeur dans la console.
