Interagir avec le document en JAVASCRIPT

JAVASCRIPT est un langage qui a la capacité d'écouter, attendre les actions du lecteur. Si le lecteur utilise sa souris, on effectue des actions sur le document WEB. De même avec le clavier.

Le script associé à cette page change les règles CSS de certains éléments HTML du document selon l'usage de la souris.

  1. Le titre change de couleur si on clique dessus.
  2. Le fond du paragraphe d'introduction change de couleur lorsque la souris l'a survolé.
  3. Le script JAVASCRIPT est de couleur bleue lorsque la souris le survole.

Les objets JAVASCRIPT utilisés sont des événements.
La syntaxe est la suivante : sélecteur.addEventListener('action', fonction à exécuter);

La fonction appelée modifiera les règles CSS du sélecteur appelé.

Le script JAVASCRIPT associé à cette page est donné ci-dessous:

Le code est donné ci-dessous:

<script type="text/javascript">
function evenements(){
    let monTitre = document.getElementById("titre");
    monTitre.addEventListener('click',changeCouleurTitre);

    function changeCouleurTitre(event){
        monTitre.style.color = "red";
    }

    let monIntro = document.getElementById("intro");
    monIntro.addEventListener('mouseover',changeCouleurFond);

    function changeCouleurFond(event){
        monIntro.style.background = "#aaa";
    }

    let monScript = document.getElementById("script");
    monScript.addEventListener('mouseover',changeCouleurTexte);
    monScript.addEventListener('mouseout',restaureCouleurTexte);

    function changeCouleurTexte(event){
        monScript.style.color = "blue";
    }
    function restaureCouleurTexte(event){
        monScript.style.color = "#4E4E4E";
    }
}
</script>

Commentaires:

  • On repère l'élément HTML sur lequel on veut agir. On attribue un identifiant s'il n'en a pas.
  • Dans le script, on applique la methode addEventListener() sur l'identifiant HTML.
  • Il faut préciser la nature du déclencheur de l'action (click de souris) et la fonction à appliquer.
  • On crée la fonction qui modifie les règles CSS de l'élément HTML ciblé.
  • On n'oublie pas de lancer l'exécution du script à la fin du document WEB.
