Votre premier script JAVASCRIPT

Il existe une autre méthode moins intrusive qui permet d'afficher des messages sans interrompre le chargement de la page web, sans intervention du lecteur. Cette méthode est console.log et s'applique sur l'objet console du navigateur. La console se trouve dans outils de développement web du menu outils du navigateur dont l'accès se fait avec la touche F12.

La console est un objet JAVASCRIPT qui possède plusieurs méthodes. La méthode log affiche des messages dans la console. Très utile lorsqu'on développe un script surtout pour connaître les valeurs des variables et détecter d'éventuelles erreurs de programmation.

Votre travail:

  1. Vous devez éditer le fichier script02.html et modifier le script JAVASCRIPT pour afficher le message "Je programme en JAVASCRIPT" dans la console.
  2. Vérifier son bon fonctionnement en ouvrant la console F12.

Si votre script est correct, en actualisant cette page, vous devez voir votre message dans la console.
