Aire d'un rectangle

Le script de cette page doit calculer l'aire d'un rectangle dont les dimensions sont saisies par le lecteur. On suppose que les valeurs sont correctement saisies et ne sont pas contrôlées.

Votre travail:

  1. Vous devez éditer ce fichier script04.html et compléter le script JAVASCRIPT pour calculer l'aire d'un rectangle.
  2. La largeur et la longueur du rectangle seront à saisir par le lecteur en 2 fois.
  3. La valeur finale sera affichée dans la console.

Si votre script est correct, il doit ressembler à celui-ci:

<script type="text/javascript">
let longueur = prompt('Quelle est la longueur du rectangle ?');
console.log('La longueur du rectangle est : '+ longueur);
let largeur = prompt('Quelle est la largeur du rectangle ?');
console.log('La largeur du rectangle estt : '+ largeur);
let aire = longueur * largeur;
console.log("L'aire du rectangle est " + aire);
</script>
