Le premier script JAVASCRIPT

Le premier script JAVASCRIPT a été exécuté à l'ouverture de ce document web.

La première fonction utilisée est alert. C'est une méthode de l'objet window du navigateur. Parmi les méthodes de l'objet window, on a :

  1. prompt() qui ouvre une fenêtre de saisie;
  2. close() qui ferme l'onglet, la fenêtre du navigateur;
  3. confirm() qui ouvre une fenêtre avec un messge;

La méthode alert() ouvre une fenêtre dite nodale pour afficher un message. Ici, son appel se fait avant l'affichage de cette page web. C'est une méthode très instrusive qui nécessite l'intervention du lecteur pour afficher la page web.

Le script de cette page est très simple et tient en une ligne. On le donne ci-dessous:

<script type="text/javascript">
window.alert('Je suis votre premier script JAVASCRIPT');
</script>
