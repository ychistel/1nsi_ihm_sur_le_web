JAVASCRIPT est un langage de programmation

Comme tout langage de programmation, JAVASCRIPT permet d'utiliser des variables et des instructions d'affectation, de condition et d'itération.

L'affectation se fait avec le signe "=" et le mot clé let : let a = 3

Les variables sont typées : number (nombres), string (caractères), array (tableaux), object (objets), etc.

Il existe un objet math en JAVASCRIPT qui permet de réaliser des calculs mathématiques.

Le script de cette page calcule l'aire d'un disque qui est affiché dans la console (F12)!

<script type="text/javascript">
let rayon = prompt('Quel est le rayon du disque ?');
console.log('Le rayon du disque que vous avez saisi est : '+rayon);
let aire = Math.PI * rayon * rayon;
console.log("L'aire d'un disque de rayon "+rayon+" est "+aire);
</script>

Commentaires:

  • Une fenêtre de saisie permet de donner le rayon du disque avec la fonction prompt.
  • Le rayon est enregistré dans une variable rayon avec la commande let.
  • Une variable aire enregistre la valeur calculée de l'aire du disque.
  • Un affichage en console est réalisé.
  • La concaténation des messages et des variables se fait avec le signe +.
