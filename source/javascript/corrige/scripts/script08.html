La boucle for en JAVASCRIPT

Le script de cette page doit demander un nombre entier positif et afficher dans la console la somme de tous les nombres entiers positifs qui lui sont strictement inférieurs.

Votre travail:

  1. Vous devez ouvrir ce fichier script08.html et compléter le script JAVASCRIPT.
  2. Le nombre sera à saisir par le lecteur.
  3. Une variable somme sera initialisée à 0.
  4. Une boucle réalisera l'addition de la somme et du compteur de boucle.
  5. La somme sera affichée dans la console.

Si votre script est correct, il doit ressembler à celui-ci:

<script type="text/javascript">
let somme = 0;
let nombre = prompt("Saisir un nombre entier pour calculer la somme de tous les nombres entiers inférieurs ou égaux.");
for(let i=1; i<=nombre; i++){
    somme=somme+i;
}
console.log("La somme des " + nombre + " nombres est égale à " + somme);
</script>

Pour approfondir

Réaliser la somme des nombres pairs puis celle des nombres impairs.
