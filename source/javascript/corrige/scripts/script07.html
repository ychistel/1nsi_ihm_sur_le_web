La strucutre itérative en JAVASCRIPT

Une structure itérative en JAVASCRIPT est une structure de boucle. Parmi les différentes possibilités, on a la boucle for:
for (let pas=0 ; pas<10 ; pas++) { instructions; } où pas est une variable compteur de boucle.

Le script de cette page affiche les différents scripts que vous avez lu pour arriver jusqu'ici.

<script type="text/javascript">
let msg="";
let valeur = prompt("Combien de scripts avez-vous lu jusqu'ici ?");
for (let i=1; i<=valeur ; i++){
    msg = msg+'vous avez lu le script '+i+'\n';
}
window.console.log(msg);
</script>

Commentaires:

  • Une fenêtre de saisie permet de donner le nombre fois que la boucle s'exécute avec la fonction prompt.
  • La valeur est enregistrée dans une variable valeur avec la commande let.
  • La boucle utilise la variable i comme compteur de boucle. Elle s'incrémente de 1 unité jusqu'à la valeur donnée au début du script.
  • On initialise la variable msg avec un caractère vide puis on concatène le message avec + dans la boucle.
  • Le message est affiché dans la console (F12).
