La fonction en JAVASCRIPT

Le script de cette page doit demander le prénom du lecteur et afficher dans la console un message de bienvenue avec une fonction.

Votre travail:

  1. Vous devez éditer ce fichier script10.html et compléter le script JAVASCRIPT.
  2. Le prénom sera à saisir par le lecteur et affecté à une variable prenom.
  3. Une fonction bienvenue sera créée avec un paramètre pnm.
  4. La fonction doit retourner le message de bienvenue avec le prénom du lecteur.
  5. Le message sera affiché dans la console.

Si votre script est correct, il doit ressembler à celui-ci:

<script type="text/javascript">
function bienvenue(prenom){
    console.log(prenom + " est bienvenu.e sur cette page.");
}
let prenom = prompt("Quel est votre prénom ?");
bienvenue(prenom);
</script>

Pour approfondir

Modifier votre fonction pour éviter un message sans prénom.
