Modifier le contenu d'un document HTML avec JAVASCRIPT

Le script de cette page doit demander le prénom du lecteur et afficher dans le document web le prénom du lecteur sous forme d'un message de bienvenue.

Votre travail:

  1. Vous devez ouvrir ce fichier script12.html et compléter le script JAVASCRIPT.
  2. Une fonction bienvenue() sans paramètre sera créée.
  3. Le prénom sera à saisir par le lecteur et affecté à une variable prenom.
  4. Le message de bienvenue est constitué avec le prénom du lecteur.
  5. Le message sera affiché dans le document WEB dans l'élément HTML "p" avec l'identifiant "bienvenue".

Si votre script est correct, en actualisant cette page, vous devez voir votre message de bienvenue écrit ci-dessous.

Pour approfondir

Ajouter l'âge du lecteur.
