Structure if ... else ... en JAVASCRIPT

Le script de cette page doit demander un nombre et afficher un message précisant s'il est négatif ou positif.

Votre travail:

  1. Vous devez éditer ce fichier script06.html et compléter le script JAVASCRIPT.
  2. Le nombre sera saisi par le lecteur dans une fenêtre avec prompt.
  3. Un message donnant le signe (négatif ou positif) du nombre sera affiché dans la console.

Si votre script est correct, il doit ressembler à celui-ci:

<script type="text/javascript">
let nombre = prompt("Saisir un nombre");
if (nombre<0){
    console.log("Le nombre "+nombre+" est négatif.");
} else {
    console.log("Le nombre "+nombre+" est positif.");
}
</script>
