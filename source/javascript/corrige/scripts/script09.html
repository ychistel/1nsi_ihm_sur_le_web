Les fonctions en JAVASCRIPT

La fonction en JAVASCRIPT est une partie de programme réalisant une tache simple qui peut être appelée à tout moment dans un programme. Les fonctions permettent de décomposer un programme en sous parties.

Comme en python, les fonctions JAVASCRIPT peuvent avoir des paramètres et retourner une valeur. La déclaration se fait avec le mot clé function et la structure est la suivante :
function nom de la fonction ( paramètres) { instructions; }

Le script de cette page utilise une fonction pour calculer le carré d'un nombre saisi par le lecteur de la page.

<script type="text/javascript">
function carré(n){
    return n * n ;
}
let nombre = prompt("Saisir un nombre dont vous voulez le carré.");
monCarré = carré(nombre);
console.log("Le carré de "+nombre+" est "+monCarré);
</script>

Commentaires:

  • Une fenêtre de saisie permet de donner le nombre dont on veut calculer le carré avec la fonction prompt.
  • La valeur est enregistrée dans une variable nombre avec la commande let.
  • La fonction carré utilise le nombre en paramètre n et retourne le calcul du carré.
  • La variable monCarré est initialisée avec la valeur retournée par la fonction carré.
  • Le carré du nombre saisi est affiché dans la console.
